# Contributing to BrewDeck 🚀

First off, thanks for taking the time to contribute! It’s seriously awesome that you want to help make **brew deck** a better macOS Homebrew wrapper. 

Before you start hacking away, please take a moment to read through this guide to make the process smooth for everyone.

---

## 🛠️ Getting Started

To work on BrewDeck, you'll need a Mac running macOS 14.0+ and the latest version of Xcode or Xcode Command Line Tools.

1. **Fork** the repository on GitHub.
2. **Clone** your fork locally:
   ```bash
   git clone https://github.com/yourusername/BrewDeck.git
